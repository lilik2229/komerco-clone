# KomercoClone

クックパッド社のネイティブアプリ komerco(https://komer.co/)をWebサービスとして写経しました。

# 使用技術

## フロントエンド
* Angular6
* Angular-Material
* angularfire2
  
## バックエンド(モック)
* Firebase Authentication
  + ユーザ認証、ユーザ管理
* Firebase Hosting
  + 静的コンテンツ(HTML,CSS,JS)の配信
* Cloud Firestore
  + データベース
* Cloud Storage for Firebase
  + 画像データの管理
